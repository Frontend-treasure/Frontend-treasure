# VitePress 语法

VitePress 兼容常规的 `Markdown` 语法 , 除此之外还拓展了一些语法, 还支持导入 Vue 组件

## 目录

```
[[toc]]
```

[[toc]]

## 表格

```md
| Tables        |      Are      |  Cool |
| ------------- | :-----------: | ----: |
| col 3 is      | right-aligned | $1600 |
| col 2 is      |   centered    |   $12 |
| zebra stripes |   are neat    |    $1 |
```

## 自定容器

```
::: info
This is an info box.
:::

::: tip
This is a tip.
:::

::: warning
This is a warning.
:::

::: danger
This is a dangerous warning.
:::

::: details
This is a details block.
:::
```
