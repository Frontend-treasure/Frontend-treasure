## ✨ 详情

> **这个网站整理了前端开发和面试相关的笔记**

**前端宝藏之站为初学者提供从基础到高级的 web 前端编程知识，而且网站上的所有内容全部免费开源!**

## 🤖 运行

```bash
# 克隆仓库
git clone

# 安装依赖
pnpm i

# 本地开发
yarn docs:dev

# 打包构建
yarn docs:build

# 启动服务
yarn docs:serve

```
