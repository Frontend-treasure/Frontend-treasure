## ✨ 详情

> **这个网站整理了前端开发和面试相关的笔记**

**前端宝藏之站为初学者提供从基础到高级的 web 前端编程知识，而且网站上的所有内容全部免费开源!**

## 🤖 运行

```bash
# clone the repo
git clone

# install the dependency
pnpm i

# start the serve
yarn docs:dev

# build
yarn docs:build

# run
yarn docs:serve

```
